HOSTINGER UPLOAD INSTRUCTIONS

This website is built as a simple static portfolio for Hostinger Premium Shared Hosting.

No React.
No Vite.
No TanStack.
No Node.js.
No npm install.
No build command.
No server.js.

UPLOAD STEPS
1. Open Hostinger hPanel.
2. Go to Websites > Manage > File Manager.
3. Open public_html.
4. Upload all files from this folder or upload the ZIP and extract it.
5. Make sure index.html is directly inside public_html.
6. Make sure assets, robots.txt, sitemap.xml, and .htaccess are also inside public_html.
7. Open https://abidbadar.online/ and test the website.
